# Agent instructions

Read `CLAUDE.md` and the relevant files in `docs/` before changing this repository.

- This is a public repository. Never publish private contact information, archive contents, applications, recommendation letters, or local video files.
- Edit canonical content in `data/`, templates in `templates/`, and assets in `static/`.
- Never hand-edit generated files in `site/`.
- Never name an under-review journal on the public site.
- Never use em dashes in public site copy.
- Run `python3 build.py` after changes and commit the regenerated `site/` output.
